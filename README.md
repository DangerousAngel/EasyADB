# EasyADB

**EasyADB** is an interactive menu-driven Python tool that simplifies Android Debug Bridge (ADB) operations through an easy-to-use command-line interface.

## Features

* 📂 Pull and push files
* 📦 Install and uninstall APKs
* 📱 Manage apps (clear data, force stop, start activities)
* 📸 Take screenshots and screen recordings
* 🔋 View battery and device information
* 📡 Monitor network connections
* 🔍 Search files on the device
* 📝 View logcat logs
* 💾 Backup Android apps and decrypt `.ab` backups
* ☎️ Extract phone numbers from contacts
* ⚡ Execute custom ADB shell commands

## Requirements

* Python 3.8+
* Android ADB installed and available in PATH
* USB debugging enabled on the Android device

## Installation

```bash
git clone https://github.com/yourusername/EasyADB.git
cd EasyADB
python EasyADB.py
```

## Usage

Connect your Android device and run:

```bash
python EasyADB.py
```

Select the desired operation from the interactive menu.

## Disclaimer

Some features may require root access or additional device permissions depending on the Android version and manufacturer restrictions.
